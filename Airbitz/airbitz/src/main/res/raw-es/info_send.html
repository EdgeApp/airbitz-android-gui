Escanear y enviar Bitcoin

Esta pantalla le permite escanear un código QR para enviar bitcoin o importar una clave privada en su cartera.

Puede escanear un código QR BitID para iniciar sesión en un sitio web usando BitID.

Puede usar Edge login a otra App decentralizada como Augur al escanear el código QR.

Elija una cartera de su cuenta haciendo clic en el nombre de la cartera en la cabecera superior.

Escanée el código QR de otra cartera para obtener una dirección para enviar fondos.

Pulse el botón [Photos] para elegir una foto que contenga un código QR que desea escanear.

Toque [Address] para escribir manualmente o pegue una dirección para enviar dinero.

Si hay solicitudes de pago de bluetooth cerca de usted, se mostraran en la parte inferior. Pulse sobre una solicitud para enviar a ese destinatario.

Toque [Transfer] (Transferir) para transferir bitcoin a otra cartera en su cuenta. Simplemente pulse el nombre de la cartera de destino en la ventana emergente.
