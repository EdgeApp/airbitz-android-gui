Información de la cartera de exportación

Elegir una carpeta para exportar usando el botón de encabezado superior.

Seleccionar [CSV] para exportar una lista CSV de todas las transacciones para su cartera seleccionada.

Seleccionar [Quickbooks] para exportar un archivo de formato QBO para importar en Quickbooks.

Seleccionar [Semilla de Cartera Privada] para exportar la semilla privada de la cartera seleccionada.
Seleccionar [Semilla de Cartera Pública] para exportar la semilla pública (XPub) de la cartera seleccionada.
AVISO, La semilla privada otorga acceso a todos los fondos en la cartera seleccionada. Mantenga la semilla privada segura y no lo exponga a los demás.
