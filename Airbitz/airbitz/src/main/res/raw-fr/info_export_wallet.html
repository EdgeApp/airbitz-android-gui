Exporter Info du Porte-monnaie


Choisir un porte-monnaie en utilisant le bouton d’en-tête de haut.

Sélectionner [CSV] pour exporter une liste CSV de toutes les transactions pour votre porte-monnaie sélectionné.

Sélectionner [Quickbooks] pour exporter un fichier de format QBO pour importer dans Quickbooks.

Sélectionner [Wallet Private Seed] pour exporter le capital privé du porte-monnaie sélectionné.
Sélectionner [Wallet Public Seed] pour exporter lae capital publique (XPub) du portefeuille sélectionné.
ATTENTION, le capital privée donne accès à tous les fonds sur le portefeuille sélectionné. Garder le capital privée sécurisé et ne l’exposez pas à d'autres.
