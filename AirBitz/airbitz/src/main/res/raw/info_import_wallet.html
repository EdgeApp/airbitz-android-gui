Import Wallet Info


This screen will import a Bitcoin private key and sweep the funds into the currently selected wallet. Choose the destination wallet by tapping the top drop down button and selecting an available wallet.

Private key import can be accomplished in several methods:
  • Scan a private key QR code using the camera
  • Type or paste the private key into the [Private Key] text field
  • Tap the Photo Album button and select a photo which contains a private key QR code
